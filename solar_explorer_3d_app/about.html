About Solar Explorer

Solar Explorer is an interactive 3D learning website designed to help users understand the relationship between the Sun, Earth and Moon through visual models, simple animation and user-controlled interaction.

Project Overview

This project has created a miniature, three-dimensional space learning environment based on the sun, earth, and moon, allowing users to go beyond simply reading text or viewing static illustrations. Instead, they can rotate the models, switch between different system views, and observe the movements of celestial bodies relative to each other.

The website focuses on several key astronomical concepts, such as the Earth's rotation, its orbit around the sun, the Moon's orbit around the Earth, the direction of sunlight, lunar phases, and basic space exploration. Its aim is to help users better understand these concepts through interactive 3D visualization technology.

Main Features

  • Individual 3D models: Users can view the Sun, Earth and Moon separately. Each object includes a short science-based introduction.
  • Sun-Earth system: This view shows Earth orbiting the Sun and includes Earth’s axial tilt to support the explanation of seasons.
  • Earth-Moon system: This view shows the Moon orbiting Earth with a fixed sunlight direction. The yellow arrows show where sunlight comes from, helping users understand why Moon phases happen.
  • Sun-Earth-Moon system: This view combines the three objects together and shows how Earth’s orbit and the Moon’s orbit happen at the same time.
  • Rocket launch animation: The Earth-Moon system includes a one-time rocket launch animation inspired by Apollo 11, showing the idea of travelling from Earth toward the Moon.
  • Interactive controls: Users can rotate or stop the animation, change camera view, switch lighting and use wireframe mode to inspect the models more clearly.

Educational Purpose

The website aims to provide an easy educational platform to help users learn basic astronomy knowledge. Many astronomical concepts are difficult to understand in words alone because they involve motion, direction, and spatial relationships. The 3D environment allows users to observe these relationships more intuitively.

For example, the Sun-Earth system can show that seasonal changes are mainly caused by the tilt of the Earth's axis of rotation, rather than determined by the distance between the Earth and the Sun. The Earth-Moon system can show that the moon phase changes are usually caused by changes in the visible part of the moon exposed to sunlight, rather than by the earth obscuring the moon.

Design and Interaction

The interface is organised to support exploration. Model selection is placed on the left side, allowing users to quickly switch between objects and systems. The 3D viewer is placed in the centre so the model remains the main focus. Control buttons are placed on the right side, making it clear how users can interact with the scene.

The background uses a dark star-field style to match the space theme. The information panel is placed below the 3D viewer so users can first explore the model and then read the related explanation.

Simplified Scale

The sizes and distances in this website are simplified for learning purposes. In real space, the distances between the Sun, Earth and Moon are extremely large compared with their sizes. If the real scale were used directly, the objects would be too far apart to view clearly on a small webpage.

For this reason, the models are arranged in a simplified way so users can clearly see the orbital relationships, light direction and movement patterns. This makes the website easier to use as a visual learning tool.

Technologies Used

The project is built using HTML, CSS and JavaScript. Bootstrap is used for the basic responsive page layout and navigation. Three.js is used to display and animate the 3D models in the browser. Blender is used to create or prepare the 3D objects before exporting them for use on the website.

These technologies work together to create an interactive browser-based 3D learning experience without requiring users to install extra software.

Deeper Understanding Statement

This project demonstrates deeper understanding by combining 3D modelling, browser-based rendering, JavaScript interaction and educational content design. It is not only a static model viewer. Users can switch between different celestial objects and system views, control animation, change lighting, inspect models in wireframe mode and view the scene from different camera angles.

The project also shows an understanding of how 3D assets are prepared for web use. The models were exported as .glb files so they could be loaded into the browser using Three.js. The scene uses lighting, camera positioning, animation loops and object grouping to create different system views, such as the Sun-Earth system and the Earth-Moon system.

The scale and orbit speeds are simplified because a fully accurate astronomical scale would not be practical in a browser-based educational website. This design choice makes the model easier to view while still helping users understand the main spatial relationships between the Sun, Earth and Moon.

Project Limitations

This project uses simplified movement and simplified scale. The orbit speeds, distances and object sizes are adjusted for visual clarity rather than exact scientific accuracy. The model is designed for learning and demonstration, not for professional astronomy calculation.
